# DAPT Testing

Testing for DAP is done using [pytest](pytest.org) and can be installed by entering `pip install -U pytest` in the terminal.  

## Command line arguments
| arg | options | usage |
| ---- |:----:| ----:|
| --color | [yes, no] | Turn terminal coloring on or off |
